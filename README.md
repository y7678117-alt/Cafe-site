# ☕ Cafe-site  

木のぬくもりと香ばしいコーヒーの香りに包まれる、  
**街角の隠れ家カフェサイト**をイメージしたWeb制作作品です。  

---

## 🌐 公開URL
➡️ [https://y7678117-alt.github.io/Cafe-site/](https://y7678117-alt.github.io/Cafe-site/)

---

## 🏠 プロジェクト概要
- **目的**：HTML/CSS/JavaScript の学習成果として、架空のカフェサイトを制作  
- **特徴**：
  - トップページ（`index.html`）：カフェ紹介・アクセス情報  
  - メニューページ（`menu.html`）：ドリンクメニュー一覧  
  - スムーズスクロールやアンカーリンクによる自然な遷移  
  - シンプルかつ温かみのある配色（ブランドカラー：`#fe8602`）  
  - PC・スマホ対応のレスポンシブデザイン  
  - アクセシビリティを意識した `alt`・`aria-label` の記述  

---

## 🧩 使用技術
| カテゴリ | 内容 |
